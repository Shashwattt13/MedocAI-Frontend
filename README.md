# 🧠 MedDocAI — Frontend

MedDocAI is a simple, privacy-focused web app that helps users understand medical documents such as prescriptions, lab reports, and scanned notes using AI.

This repository contains the **frontend** of the MedDocAI project.

---

## ✨ What This App Does

- Upload up to **3 medical documents**
- Shows a clear **AI analyzing status**
- Displays explanations in **easy, human language**
- Allows users to **download a structured medical PDF**
- No login, no tracking, no data storage

---

## 🛠️ Built Using

- React.js
- Vite
- Tailwind CSS
- React Router

---

## 🔐 Privacy Notice

Uploaded files are processed temporarily and are not stored.
